Model & Floor

Build the production line and lay it out in two dimensions — where things sit changes how the line performs.

Coming in Phase 3

2D layout & transport canvas

A quiet, diagrammatic canvas to drag-place resources and storage, with transport derived from 2D distance: placement sets distances; distances set transport delays; movers are limited resources (a conveyor on a fixed path, and people/worker movers with a count and speed). Deliberately simple — no pathfinding, AGV dispatch, collisions, or auto-layout — so the link between layout and flow stays legible.
