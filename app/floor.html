Model & Floor

Place resources and storage on the floor and connect them into a route. Travel time comes from the distance between placed nodes, so where things sit changes how the line performs. Click any node or any transport leg to set its parameters.

Single linear route for now. Each station takes a full service-time distribution; each transport leg can be set independently. Branching and assembly come later.

Pick a tool, then click the canvas to place a node. In Move, drag a node to reposition; click a node or a leg to edit it on the right. The route follows the order nodes are added.
