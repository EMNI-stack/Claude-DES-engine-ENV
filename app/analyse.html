Step 03 · the rigour layer

Run & Analyse

A single run is one sample path of a random process — never "the answer". Run the model several times with different random seeds and read every response as a mean with a confidence interval, so the uncertainty is always in view.

Replications

Run the model honestly

Each replication is an independent run with its own random seed (seeds 1…10). More replications → a tighter interval (the precision improves with √N).

Warm-up handling for steady-state studies (deciding how much of the start to delete) arrives in the next step. For now every replication is measured over its whole length.
