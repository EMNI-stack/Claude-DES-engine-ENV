System & Production Line Lab

Build a system from stations connected in a line. Each station has a buffer, parallel machines, and a service-time distribution. Server systems lose customers when a buffer is full; production systems block upstream machines and can scrap parts or break down. Configure any distribution, add extra variability, then watch parts flow through the system you designed.
